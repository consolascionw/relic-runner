# Game Title: Homestead Adventure

## Game Title & Description
**Cute Fantasy RPG: Homestead Adventure** is a vibrant top-down adventure and farming simulator. Players are tasked to build a wonderful world, manage their own farm, and construct a cozy house while exploring a 16-bit fantasy landscape.

The core loop involves gathering resources, tilling the land, and defending the farm from whimsical enemies like Slimes and Bombshrooms. It combines the tranquility of farming with the thrill of exploration and combat.

## Controls
- **WASD / Arrow Keys** - Move Player
- **Left Click / J** - Primary Action (Attack / Till / Chop / Mine)
- **Right Click / K** - Secondary Action (Watering / Using Items)
- **Space** - Roll / Dash
- **I** - Open Inventory
- **Esc** - Pause Menu

## How to Run the Game
- **Godot Version:** Godot 4.5-stable
- **Steps to Run:**
  1. Open Godot
  2. Import project
  3. Press Play

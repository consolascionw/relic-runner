# Game Title: Dark Continent

## Description
Dark Continent is a 2D action platformer where players explore dark environments, defeat enemies, and unlock new areas.

The core gameplay focuses on movement, combat, and exploration using a dark knight character.

## Controls
Q - Dash
W / Up Arrow - Jump
A / Left Arrow - Move Left
D / Right Arrow - Move Right

Esc / P - Pause

## How to Run the Game
Godot Version: Godot 4.2

Steps:
1. Open Godot
2. Click Import
3. Select project folder
4. Press Play

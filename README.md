# Chrome Gerson Run

# Game Description
Gerson Run is a level based running game where the player
controls Gerson that runs forward and must reach a specific distance
goal to finish each level. This is vertical scrolling

The game is designed to be [fast-paced / strategic / casual / challenging],
with an emphasis on [movement, timing, exploration, combat, etc.].

Controls
Keyboard / Mouse

W / A / S / D or Arrow Keys – Move
Space – Dash
Esc – Pause

How to Run the Game

Godot Version Used: Godot 4.2

 Running from Godot Editor
1. Open - Godot Engine
2. Click - Import
3. Select the project folder
4. Press - Play
